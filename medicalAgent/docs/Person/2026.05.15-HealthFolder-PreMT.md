## Notes on MyChart / Epic FHIR Access

Frank's provider (ARC) runs Epic MyChart. Three access paths are documented in `design_1.0_medicalTracker.md`:

1. **Native Bulk Export** — manual EHI/Lucy Summary download (XML or FHIR JSON); requires clicking, takes hours/days
2. **FHIR API** — best for real JSON; requires manual re-auth each session due to Epic's 2FA enforcement
3. **Playwright Scraping** — true automation; fragile against Epic DOM updates
